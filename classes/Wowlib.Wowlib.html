    Parameters

    • The array key for this realm ID. Can be a stringname, or Integer, and is used only for when you need to use the getRealm() method.

    $DB   [Default = array()]

    An array of database connection information as defined below. Not needed unless loading a new realm that is previously unloaded. array( 'driver' 'host' 'port' 'database' 'username' 'password' )
