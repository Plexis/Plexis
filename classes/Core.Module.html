    install(bool $override = false) : bool

    Parameters

    $override   [Default = false]

    bool

    Remove conflicting routes from other modules? If false, and there is a routing conflict, an \Exception will be thrown. Otherwise, the old route will be removed, and the new inserted.

    Exceptions

    \Exception Thrown if there is a routing conflict with another module, and $override is set to false.

    Returns

    boolReturns true on success, false otherwise
